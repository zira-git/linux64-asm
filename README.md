# linux64-asm
- ### Assembly x64 examples for Linux with NASM syntax
- ### The examples are for learning or re-use

### Current Examples:
- ### Hello, World! (hello_world.asm)

### Running ASM
- ### Compile with NASM: nasm -f elf64 filename.asm -o filename.o
- ### Link it with GCC: gcc filename.o -o filename -nostdlib -static
- ### And you will get a nice executable file that can be ran with: ./filename
- ###### MAKE SURE YOU HAVE GCC AND NASM INSTALLED!
